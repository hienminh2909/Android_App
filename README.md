Android Development Practice Projects
**Overview:**
  - This repository contains a collection of Android application exercises focusing on core mobile development concepts such as UI components, notifications, and cloud integration.
  => Goal: Build foundational skills in Android development through hands-on mini projects.

**Project Structure**
/ListView_Music        → Display music list using ListView
/Notification          → Implement Android notification system
/Temp_Humi_Firebase    → Firebase integration (real-time data) + Raspberry + Arduino.
/Parking               → Firebase integration ESP32 with module RFID RC522.
/Control_Device_SIM900 → Integrate with Module SIM900 and Arduino Uno to control servo 

**Key Features**
- Android UI development
- Notification system implementation
- Firebase and MCU integration (real-time database, IOT system)
- Modular mini-project structure
- Practice-based learning approach
